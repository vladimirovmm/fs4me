# Изменения в README.md

## Добавлено

### 1. Типы и структуры (🧩)
- `Stat` — информация о файле/директории
- `DriverParams` — параметры подключения (с примерами создания из HashMap, строки, C-строки)
- `WriteMode` — режимы записи (overwrite, append, fail-if-exists)
- `LockMode` — режимы блокировок (read, write)
- `FileKind` — типы файлов (file, dir, link)
- `Read` — чтение файла с SeekPosition
- `Write` — запись в файл

### 2. Использование пакетов (📦)
- `fs4me-interface` — базовый интерфейс (trait Driver)
- `fs4me-local` — локальный драйвер
- `fs4me-client` — обёртка с lock-механизмами и UUID

### 3. FFI-экспорт (🔧)
- Обёртки для C-программ (fs4me_connect, fs4me_info, fs4me_read, fs4me_write)
- Примеры использования в C-проекте

### 4. Модуль `ffi_tests` (🧪)
- Примеры тестов FFI в C

### 5. Структура проекта (📁)
- Полная диаграмма проекта

### 6. Примеры кода
- Rust-примеры для каждого пакета
- C-пример для FFI-экспорта

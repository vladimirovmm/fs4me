# fs4me-lock

Библиотека для межпроцессных блокировок с UUID.

## Как работает

- Атомарная операция `mv` обеспечивает эксклюзивный доступ к файлу блокировки
- Формат записи: `FsUuid=unixtime=mode` — UUID, время, режим блокировки
- Один процесс держит только одну блокировку
- При падении процесса (`Drop`) блокировка автоматически снимается

## Типы блокировок

- **Read** — много процессов одновременно
- **Write** — один процесс, блокировка других
- **WriteQueue** — в очередь на запись, первый в очереди получает блокировку

## Использование

```rust
let lock = MultiLock::try_from(uuid, driver, path, LockMode::Read)?;
// при выходе из области видимости блокировка снимается
```
